# SereniMind 🧠 
**Your Personal Mental Wellness Companion**

## About The Project
SereniMind is a frontend web application developed as part of the BCSE203E Web Programming curriculum. It is designed to provide students with a localized, offline-capable digital space to practice guided breathing, manage focus, and track daily emotional states. 

## Key Features
* **🫁 Guided Breathing:** Interactive CSS/JS animations for timed Inhale/Hold/Exhale cycles.
* **⏱️ Focus Timer:** A built-in Pomodoro timer with dynamic SVG progress rings and Web Audio API notifications.
* **💭 Mood Tracker:** Daily emotional logging with `localStorage` persistence and weekly average calculations.
* **✨ Affirmation Generator:** Randomized, non-repeating positive affirmations.

## Technologies Used
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Glassmorphism UI, Custom Properties (Variables), CSS Grid/Flexbox, and Keyframe Animations.
* **Vanilla JavaScript:** DOM manipulation, State Management, Web Audio API, and LocalStorage API.
